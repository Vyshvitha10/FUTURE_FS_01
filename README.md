# Web Development Project Collection
## Personal Portfolio + Mini CRM + Local Business Website & Live Pitch

## Project Overview

This repository contains three complete web development projects designed to demonstrate frontend development, backend integration, database management, user experience design, and real-world business problem solving.

The collection includes:

1. Personal Portfolio Website
2. Client Lead Management System (Mini CRM)
3. Local Business Website & Live Pitch Project

Together, these projects showcase practical development skills, responsive design, application architecture, business understanding, and deployment workflows.

# Project 1 — Personal Portfolio Website

## Description
A modern and responsive personal portfolio website developed to present professional information, technical skills, featured projects, achievements, and contact details.

The portfolio acts as a personal brand platform and enables visitors to explore projects and connect professionally.

### Objectives
- Create a professional digital identity
- Showcase skills and achievements
- Improve visibility through SEO
- Build an interactive user experience

### Features
✔ Responsive design for all devices  
✔ Interactive hero section  
✔ About and Resume section  
✔ Skills showcase  
✔ Project gallery  
✔ Contact form  
✔ Email integration (optional)  
✔ SEO optimization  
✔ Smooth navigation and animations  

### Technologies Used

#### Frontend
- HTML5
- CSS3
- JavaScript
- React.js

#### Backend (Optional)
- Node.js
- Express.js

#### Database (Optional)
- MongoDB / MySQL

# Project 2 — Client Lead Management System (Mini CRM)

## Description
A lightweight Customer Relationship Management (CRM) system developed to manage leads collected through website contact forms.

The system allows businesses to organize customer information, update lead progress, maintain notes, and improve follow-up processes.

### Objectives
- Store and manage client leads
- Track customer communication
- Learn CRUD operations
- Implement backend and database integration

### Features
✔ Add client leads  
✔ View lead dashboard  
✔ Edit and update lead information  
✔ Lead status tracking  
✔ Notes and follow-up management  
✔ Secure admin authentication  
✔ REST API integration  

### Lead Workflow

New → Contacted → Converted

### Technologies Used

#### Frontend
- HTML5
- CSS3
- JavaScript
- React.js

#### Backend
- Node.js
- Express.js

#### Database
- MongoDB / MySQL
